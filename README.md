# react-study

> React Tutorial with Gulp

## Usage

```
$ git clone git@github.com:jodeng110/react-study.git <my-new-project>
```

```
$ cd <my-new-project>
```

### Install

```
$ npm install -g gulp bower
```

```
$ npm install
```

```
$ bower install
```

### Run
```
$ gulp develop
```

open Web Browser : http://localhost:3000/
